# 새 프로젝트 개발 환경 자동화

신규 프로젝트에 완전 자동화된 개발 환경을 구축해줘:

🎯 목표:
- 깨끗한 환경에서 최적화된 설정
- 팀 협업 기준 확립
- TDD 워크플로우 완전 자동화

🛠️ 설치 및 구성:
1. Husky + Lint-staged 초기 설정
2. ESLint + Prettier 통합 구성
3. Jest + Testing Library 환경
4. Git 훅 자동화
5. VS Code/Cursor 통합 설정

📋 새 프로젝트 특화 요구사항:
- package.json scripts 최적화
- 초기 폴더 구조 생성
- 샘플 테스트 파일 생성
- 팀 가이드라인 문서 생성
- CI/CD 준비 설정

모든 설정을 처음부터 완벽하게 구성하고,
팀원 온보딩 가이드도 포함해줘.
